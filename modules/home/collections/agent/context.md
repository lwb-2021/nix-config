# AGENTS.md

## 交流语言

- 请始终使用 **简体中文** 与我对话、解释问题、提出建议，以及编写或更新任何 **skill**、**subagent**、**prompt**等。
- **代码和注释**必须使用 **英文**（变量名、函数名、类型、注释等）。

## 行为准则
- 在指令模糊、难以理解或者有多种理解方式的情况下，**必须**暂停操作并且进行询问和确认。

## 操作须知

- 当前操作系统为**NixOS**，使用 **Home Manager** 管理家目录下**所有** dotfiles。
- NixOS 和 Home Manager (作为NixOS模块安装) 配置位于 `~/Configurations/nix-config`中，是一个Flake配置，并通过GitHub引用了`~/Configurations`下的其他文件夹中的flake。
- **根目录(`/`)和家目录 (`~`)** 位于 **tmpfs** 上，**不要假定任何文件会持久化存储**。大部分直接写入 `~/.xxx` 或 `~/xxx` 的文件、目录、配置都会在重启后丢失。
- 当你需要持久化地添加一个新工具、修改 shell 别名、更改编辑器配置时，请首先查找并修改对应的 Nix 声明，**不要直接操作家目录中的文件**。
- **禁止** 试图跟随符号链接去修改`/nix/store`中的文件，当你需要修改时，请去修改`~/Configurations`下的配置。
- **禁止** 直接在`~`下未持久化的目录(如`.config`)写入配置文件、脚本等。
- **禁止** 直接下载和运行安装脚本，**必须**通过修改系统配置或建立devShell的方式安装**需要长期使用**的程序。
- 修改 `~/Configurations/nix-config` 中的文件或其引用的其他配置文件后，运行 `nh os switch -e pkexec`来应用更改。
- **务必** 尽可能遵守声明式原则。

## CLI工具
- 如果临时使用一个当前环境不存在的包，**可以**通过`nix run nixpkgs#package-name -- <argv>`直接运行。
- 对于运行时间较长的进程，**可以**用tmux运行。
